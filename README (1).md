# 🔵 PAN Attrition Intelligence Dashboard
### Machine Learning–Based Employee Attrition Prediction & Risk Scoring System
**Palo Alto Networks — HR Analytics Internship Project**

---

## 📁 Required Files in This Repository

```
your-repo/
├── app.py                              ← Main Streamlit app
├── requirements.txt                    ← Python dependencies
├── attrition_model.pkl                 ← Trained ML model (download from Colab)
└── Employee_Attrition_Risk_Scores.csv  ← Risk scores (download from Colab)
```

> ⚠️ **You MUST upload both `.pkl` and `.csv` files** from your Colab output into this repo.

---

## 🚀 Deploy on Streamlit Cloud (Free)

### Step 1 — Create GitHub Repository
1. Go to [github.com](https://github.com) → **New Repository**
2. Name it: `pan-attrition-dashboard` (public)
3. Upload all 4 files listed above

### Step 2 — Connect to Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Sign in with GitHub
3. Click **"New app"**
4. Select your repo → branch: `main` → Main file: `app.py`
5. Click **"Deploy"** → Your app will be live in ~2 minutes!

### Step 3 — Share Your Link
You'll get a public URL like:
`https://your-username-pan-attrition-dashboard-app-xxxx.streamlit.app`

## 📊 Dashboard Modules

| Module | Description |
|--------|-------------|
| 📊 Risk Dashboard | KPI cards, risk distribution, department overview, high-risk employee table |
| 👤 Employee Profile | Individual attrition probability, gauge chart, satisfaction radar, risk factors |
| 🏢 Department View | Role-level risk breakdown, income vs risk scatter, full team table |
| 🔬 What-If Explorer | Simulate HR interventions (salary raise, remove overtime, improve satisfaction) |
